CSS Flexbox

Flexbox is short for Flexible Box module, wherein it arranges items in rows and columns without using float or positioning. All you need to do is to wrap the div properties inside a class container then targetting it in your CSS and set display to "flex".

Flex Sizing

1. flex-grow

We look at two properties that affect the size and flexibility of flex items along the main axis: flex-grow and flex-shrink. Flex-grow shows how each flex item are displayed as growing relative to each other. As shown, the second flex item is shown as having the flex-grow property, while the others are displayed as normal.

2. flex-shrink

The flex-shrink defines how flex items should shrink compared to other items. The second flex item is shown as having the flex-shrink property, while the others are displayed as normal.

The CSS Flexbox Properties

Properties Description
display Specifies the display behavior (the type of rendering box) of an element. These are "none, inline, block, and inline-block".
flex-basis Specifies the initial length of a flex item.
flex-direction Specifies the direction of the flex items.
flex-grow Specifies how much a flex item and the positive free space will grow.
flex-shrink Specifies how much a flex item will shrink. It will show how much negative free space can be removed.
gap A shorthand property for the row-gap and the column-gap properties.
grid A shorthand property.
justify-content Horizontally aligns the entire grid within the container.

The Flex-Direction Values

Values Description
row Default value. Items are displayed horizontally.
row-reverse Items are displayed in reverse as row.
column Items are displayed vertically.
column-reverse Items are displayed in revserse as column.
