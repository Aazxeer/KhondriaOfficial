CSS Grid

CSS Grid is a layout system used in web designing to organize content in two dimensions: rows and columns. It makes it easy to design complex and responsive web pages without using positioning.

How is CSS Grid different from Flexbox?

  • Flexbox: A tool that works in one dimension (either a row or column). It's good for aligning items in a straight line.
  • CSS Grid: A tool that works in two dimensions (rows and columns). It's better for creating layouts such as grids or tables.

A grid layout contains of a parent element (the grid container) and the direct children of the grid container as the grid items.

The div element becomes a grid container when its display property is set to grid or inline-grid.

Display

Using the display:grid will make the grid container a block-level.

The CSS Grid Properties

Properties Description
column-gap Specifies gap between columns.
display Specifies the display behavior (the type of rendering box) of an element. These are "none, inline, block, and inline-block".
gap A shorthand property for the row-gap and the column-gap properties.
grid A shorthand property.
grid-template-columns Specifies the size and number of columns in a grid layout.
justify-content Horizontally aligns the entire grid within the container.
